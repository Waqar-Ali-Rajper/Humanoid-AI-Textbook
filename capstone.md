# capstone

Title: Capstone Project

Your final task builds a complete autonomous humanoid in simulation.

Goal
A simulated robot receives a voice command, plans, navigates, identifies an object, and manipulates it.

You build these systems

Voice interface using Whisper

Language reasoning using an LLM

Perception pipeline with cameras and depth

Navigation with Nav2

Manipulation with ROS 2 controllers

Simulation with Isaac Sim or Gazebo

Required steps

Load humanoid model

Add sensors

Build ROS 2 nodes

Add perception models

Add action planner

Test navigation

Test grasping

Connect speech input

Run full pipeline end to end

Evaluation points

Stable simulation

Clean module structure

Accurate perception

Steps generated from natural language

Safe and smooth execution

Your capstone proves your understanding of Physical AI.
